**Design details:**

- aria-label on dismiss button for accessibility
- Only shows when: not loading, clientData exists, and not previously dismissed

**Why sessionStorage (not localStorage):**
Per STATE.md decision: sessionStorage survives page refreshes but reappears on new browser sessions. This is intentional - banner is brief and helpful, so showing it again on a new visit is fine. Avoids localStorage staleness issues.

**Why clientId in storage key:**
Prevents one client's dismissal from affecting another client viewing their own quote in the same browser.
